# 云开发配置操作步骤

## 1. 配置云环境ID

已在 `app.js` 文件中配置了云环境ID为 `cloud1-3g41ctlnd6ba594a`。

## 2. 创建云数据库集合

### 创建 `checkInRecords` 集合

1. 打开微信开发者工具 → 点击左侧「云开发」按钮
2. 进入云开发控制台 → 点击左侧「数据库」
3. 点击「+ 集合」按钮 → 输入集合名称 `checkInRecords` → 点击「确定」
4. 点击 `checkInRecords` 集合 → 点击「权限设置」
5. 选择「仅创建者可读写」→ 点击「确定」

### 创建 `userInfo` 集合

1. 打开微信开发者工具 → 点击左侧「云开发」按钮
2. 进入云开发控制台 → 点击左侧「数据库」
3. 点击「+ 集合」按钮 → 输入集合名称 `userInfo` → 点击「确定」
4. 点击 `userInfo` 集合 → 点击「权限设置」
5. 选择「仅创建者可读写」→ 点击「确定」

## 3. 让开发者工具识别云函数目录

已在 `project.config.json` 中配置 `cloudfunctionRoot`: `cloudfunctions/`。  
若右键 `cloudfunctions/login` 仍无「当前云函数配置」等选项：

1. 关闭当前项目后重新用微信开发者工具打开项目根目录（含 `project.config.json` 的目录）
2. 确认左侧文件树中 `cloudfunctions` 及其子文件夹 `login`、`checkIn` 显示为云函数图标（云朵）
3. 右键 `cloudfunctions/login` 应出现「上传并部署」「当前云函数配置」等菜单

## 4. 配置 login 云函数 APP_SECRET（获取真实 openid 必做）

若云函数 `getWXContext()` 未返回 openid，login 会通过 **code2Session** 用 code 换取 openid，需提供小程序密钥。任选其一即可。

**方式 A：环境变量（推荐）**

1. 在微信开发者工具中：云开发 → 云函数 → 选中 `login` → 配置 → 环境变量
2. 新增：键 `APP_SECRET`，值为小程序 AppSecret（[微信公众平台](https://mp.weixin.qq.com) → 开发管理 → 开发设置）
3. 保存后**重新上传并部署** login 云函数

**方式 B：本地 config.json（控制台环境变量未生效时）**

1. 在项目中将 `cloudfunctions/login/config.example.json` 复制为 `cloudfunctions/login/config.json`
2. 打开 `config.json`，将 `APP_SECRET` 的值改为你的小程序 AppSecret
3. 保存后**重新上传并部署** login 云函数（上传时会把 config.json 一起打包，注意勿将 config.json 提交到 git，已加入 .gitignore）

## 5. 部署云函数

### 部署 `checkIn` 云函数

1. 打开微信开发者工具 → 点击左侧「cloudfunctions/checkIn」文件夹
2. 右键点击 `checkIn` 文件夹 → 选择「上传并部署（云端安装依赖）」
3. 等待部署完成，看到「部署成功」提示

### 部署 `login` 云函数

1. 右键点击「cloudfunctions/login」→「上传并部署（云端安装依赖）」
2. 部署完成后，若已按上文配置了 `APP_SECRET`，登录时会得到真实 openid

## 8. 常见问题解决

### 问题：获取 openid 为 undefined

1. 确认已配置 **APP_SECRET**：环境变量（云函数配置）或 `cloudfunctions/login/config.json`（复制 config.example.json 并填写密钥）
2. 确认小程序端传入了 **code** 与 **appid**（本项目已自动传入）
3. 配置或修改后必须**重新上传并部署** login 云函数
4. 若控制台环境变量不生效，请使用 config.json 方式（见上文「方式 B」）

### 问题：云函数部署失败

解决步骤：
1. 检查网络连接是否正常
2. 确保云开发环境已开通
3. 右键云函数文件夹 → 选择「上传并部署（云端安装依赖）」

### 问题：打卡失败

解决步骤：
1. 检查是否已授权位置权限
2. 确保在公司打卡范围内
3. 检查网络连接是否正常
4. 检查云函数是否已部署成功

### 问题：用户信息获取失败

解决步骤：
1. 检查是否已授权微信登录
2. 确保云开发环境已开通
3. 检查 `userInfo` 集合是否已创建
