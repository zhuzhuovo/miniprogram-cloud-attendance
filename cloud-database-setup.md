# 云数据库集合创建步骤

## 创建 `checkInRecords` 集合

1. 打开微信开发者工具 → 点击左侧「云开发」按钮
2. 进入云开发控制台 → 点击左侧「数据库」
3. 点击「+ 集合」按钮 → 输入集合名称 `checkInRecords` → 点击「确定」
4. 点击 `checkInRecords` 集合 → 点击「权限设置」
5. 选择「仅创建者可读写」→ 点击「确定」

## 创建 `userInfo` 集合

1. 打开微信开发者工具 → 点击左侧「云开发」按钮
2. 进入云开发控制台 → 点击左侧「数据库」
3. 点击「+ 集合」按钮 → 输入集合名称 `userInfo` → 点击「确定」
4. 点击 `userInfo` 集合 → 点击「权限设置」
5. 选择「仅创建者可读写」→ 点击「确定」
